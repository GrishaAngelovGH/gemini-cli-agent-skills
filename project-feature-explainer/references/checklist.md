# Feature Explanation Checklist

Before finalizing the explanation, ensure the following:

- [ ] **Context Provided:** Did you explain *why* this feature exists in the project?
- [ ] **Accurate Symbol Names:** Are all function, class, and variable names exactly as they appear in the code?
- [ ] **File Paths:** Did you include relative file paths for all mentioned components?
- [ ] **Security/Performance Notes:** Did you mention any specific security measures (e.g., encryption) or performance considerations (e.g., caching)?
- [ ] **Dependency Mapping:** Are external libraries used by this feature clearly identified?
- [ ] **Code Samples:** Is the usage example runnable or at least syntactically correct?
